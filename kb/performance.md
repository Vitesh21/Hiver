# Email Loading Performance

Slow loading can result from network, large threads, or attachment processing.

## Optimization Steps

- Use the latest browser; clear cache.
- Disable heavy extensions for the Hiver tab.
- Limit conversation length by archiving old threads.
- Avoid oversized inline images and massive attachments.
- Monitor status page for known outages.

## Troubleshooting

- Measure load times; identify specific mailboxes impacted.
- Check developer console for errors.
- Report examples with timestamps to support for analysis.
