# Tagging in Hiver

Tags categorize conversations to support workflows and analytics.

## Best Practices

- Keep tag names concise and consistent.
- Use automations to apply tags based on subject/body patterns.
- Periodically review unused tags to reduce noise.

## Troubleshooting Tag Issues

- If tags are missing, check permissions and tag visibility.
- Ensure automations applying tags are enabled and correct.
- Re-index suggestions if accuracy drifts.
